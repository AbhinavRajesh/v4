A small side project I built when Wordle was peak hype — a fully playable CLI clone of the game, built in TypeScript and published to npm.

You can install it globally and play the daily puzzle from your terminal:

```bash
npx @abhinavrajesh/wordle
```

I wrote a detailed blog post covering how I built it — picking the word list, handling color feedback in the terminal, packaging the binary, and publishing to npm. You can read it [here](/notes/how-i-built-cli-version-of-wordle-game).

### Tech stack:
- TypeScript
- Node.js
- Inquirer (interactive prompts)
- Chalk (terminal colors)
- npm (distribution)
